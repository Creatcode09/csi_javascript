Problem Statement PART A

Exchange Code Challenge

You are given a programming problem that must be solved in two connected stages. Each teammate first solves the first phase of a different problem. After the timer ends, the code is exchanged, and each teammate continues the second phase using the other teammate’s submitted Part A code.

Part A

Build the first part of the solution and write a correct intermediate logic so that your teammate can continue from your code later.

Part B

Continue the given exchanged code, complete the second phase of the logic, and produce the final correct solution.

Use the Switch Code option after the timer ends to simulate the exchange step between teammates.
